Offline HTML5 Map Application

Welcome!

Welcome to my Offline HTML5 Map Application. On this webpage I will be giving some more details about the map I have created, and how best to navigate and use it.


What the map is about

This map is designed to show you, the user, some of the different Areas of Natural Beauty. It is something that is not usually highlighted in much detail on traditional map applicaitons. So I have decided to create a map that anyone can use to find and explore some of the different natural beauty spots in the UK.


How to use the map

The map will appear as soon as you open up the website. You will be prompted with the below screen. This highlights all of the different aspects of the map.

How to move around the map

You can click and drag anywhere on the map to move around. You can also use the search bar to find a specific location. There are markers on the map which will also help with locating the place you are looking for.


How to get directions using the map
